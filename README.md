# Fullscreen Timer

This is a simple countdown timer for presentations with a separate question time stage.

![animation](docs/animate.gif)

Please try online version: https://alphakevin.github.io/fullscreen-timer/

## Keyboard Shortcut

* <kbd>F</kbd> - toggle fullscreen mode.
* <kbd>←</kbd> <kbd>→</kbd> <kbd>↑</kbd> <kbd>↓</kbd> - edit timer.
* <kbd>R</kbd> - reset timer
* <kbd>Space</kbd> - start / pause timer
* <kbd>Space</kbd> - continue to question time after presentation overtime starts

Presentation starts at 15 minutes. Question time starts at 5 minutes, and any unused presentation time can be carried into question time with the on-screen carryover toggle.
